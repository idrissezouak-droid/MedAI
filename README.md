# MedAI 🩺🤖

AI-powered medical assistant with symptom analysis, emergency detection, geolocation, multilingual support, voice input, and PDF diagnostic export.

## 🚀 Features

- 🧠 AI symptom analysis
- 🚨 Intelligent emergency detection
- 🎤 Voice symptom input
- 🌍 Multilingual support (French, English, Arabic)
- 📍 Nearby hospitals based on geolocation
- 🗺️ Dynamic emergency maps
- 📄 PDF medical diagnostic export
- 🕓 Analysis history system
- ⚡ Fast and responsive UI
- 🌙 Modern dark medical interface

## 🛠️ Technologies Used

### Frontend
- HTML5
- CSS3
- JavaScript

### Backend
- Node.js
- Express.js

### APIs & Services
- OpenRouter AI API
- Browser Geolocation API
- Google Maps Embed
- SpeechRecognition API
- jsPDF

```bash
git clone https://github.com/idrissezouak-droid/MedAI.git
cd MedAI
npm install
````

## ▶️ Run Project

### Backend

```bash
cd backend
npm install
npm start
```

### Frontend

```bash
cd frontend
npx serve .
```
